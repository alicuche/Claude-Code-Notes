Every token persists: user messages, AI output (including tool_use JSON), AND tool results — all resent every turn.
Read 1 file = 5K tokens stuck for the rest of the conversation. 10 file reads = 50K tokens resent EVERY subsequent call.
→ This is why context window matters — not “AI forgets” but “cumulative cost explodes”.
Note: Prompt caching reduces cost but does NOT reduce context window consumption.
