$ Full Debug Command

$ ANTHROPIC_LOG=debug claude \ --debug \ --verbose \ -p "how are you?" \ --output-format stream-json

? Flag Reference

ANTHROPIC_LOG=debug Anthropic SDK
Dumps full HTTP request/response to stderr
--verbose Show detailed output including tool calls
-p "prompt" Non-interactive mode — run single prompt and exit
--output-format stream-json Output
Stream NDJSON events (init, tool, assistant, result)
--debug Claude CLI
Write debug log to ~/.claude/debug/
